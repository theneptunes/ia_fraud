Existem muitos modelos classificadores que podem ser usados para detectar transações financeiras fraudulentas. Alguns dos mais comuns são:

Regressão Logística: É um modelo linear que é útil para classificar dados binários. Ele modela a probabilidade de um evento ocorrer, com base em várias variáveis independentes.

Árvore de decisão: É um modelo de classificação que divide um conjunto de dados em ramos e nós de decisão para prever uma classe ou valor de destino.

Random Forest: É um modelo de aprendizado em conjunto que usa várias árvores de decisão para classificar os dados.

Redes Neurais: São modelos de aprendizado profundo que simulam o funcionamento do cérebro humano para encontrar padrões nos dados.
